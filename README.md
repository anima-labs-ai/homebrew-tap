# Homebrew Tap for Anima

Official [Homebrew](https://brew.sh) tap for the [Anima CLI](https://github.com/anima-labs-ai/cli).

## Installation

```bash
brew tap anima-labs-ai/tap
brew install anima
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `anima` | Official CLI for the Anima platform |

## Updating

```bash
brew update
brew upgrade anima
```
